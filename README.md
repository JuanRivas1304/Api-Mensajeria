### Clonar el repositorio

git clone https://github.com/JuanRivas1304/Api-Mensajeria.git

### Instalar dependencias

pip install -r requirements.txt

### Ejecutar el servidor

python main.py

### Enviar correo

usar las rutas que estan en el main.py

### Usar el API en postman

para usar el api hay que configurar el envio como json y enviar el email_receiver o email utilizando un correo real como parametro dependiendo de la ruta que se este usando
